## 📚 Course Contents

### Day 1-5: Linux Ki Duniya Mein Pehla Kadam 🌱

* Day 1: 🐧 **Linux Kya Hai** - Linux ka history, Windows se fark, alag alag distros (Ubuntu, Fedora, Arch) aur Linux kyun seekhna chahiye — bilkul simple bhasha mein.
* Day 2: 💻 **Terminal & Basic Commands** - Terminal kya hota hai, pehli baar kaise use karte hain, aur sabse zaroori commands: `ls`, `cd`, `pwd`, `mkdir`, `touch`, `rm`, `cp`, `mv`.
* Day 3: 📁 **Linux File System** - Linux mein files kahan hoti hain? `/home`, `/etc`, `/var`, `/tmp` ka matlab. Windows jaisi C: D: drive nahi hoti — yahan sab ek hi jagah!
* Day 4: 🔐 **File Permissions** - `rwx` ka matlab kya hai, `chmod`, `chown` kaise kaam karte hain, aur numbers (755, 644) ka matlab.
* Day 5: ✏️ **Text Editors — Vim & Nano** - File mein kuch likhna hai? Nano (easy) aur Vim (powerful) dono seekho. "Vim se bahar kaise nikalein" — yeh bhi batayenge! 😄

---

### Day 6-10: System Ko Control Karna ⚙️

* Day 6: 👤 **Users & Groups** - Linux mein alag alag users hote hain. Naye user kaise banate hain, `sudo` kya hota hai, groups mein kaise add karte hain — sab seekho.
* Day 7: 📦 **Package Management** - Software install kaise karte hain? `apt` (Ubuntu), `yum`/`dnf` (CentOS), `pacman` (Arch) — yeh sab package managers hain, inhe seekho.
* Day 8: ⚡ **Processes & Jobs** - Computer mein ek saath kai programs chalte hain. `ps`, `top`, `htop`, `kill`, background/foreground jobs — sab control karna seekho.
* Day 9: 📝 **Text Processing** - `grep`, `sed`, `awk`, `cut`, `sort`, `uniq` — yeh commands text files se kaam karne ke liye hain. Pipes (`|`) ka jaadu!
* Day 10: 🔧 **Shell Scripting — Basics** - Apna pehla script likhna! Variables, if-else, loops, functions — programming Linux mein kaise hoti hai.

---

### Day 11-15: Networking & Storage 🌐

* Day 11: 🚀 **Shell Scripting — Advanced** - Arrays, error handling, real-world automation scripts, arguments (`$1`, `$2`), aur professional scripts kaise likhte hain.
* Day 12: 🌐 **Networking Basics** - `ping`, `ifconfig`, `ip`, `netstat`, `ss`, `curl`, `wget` — internet aur network se related sab commands.
* Day 13: 🔑 **SSH & Remote Access** - Doosre computer mein remotely login karna! SSH keys kaise banate hain, `scp` se files bhejte hain, aur secure connection setup karna.
* Day 14: 💾 **Disk & Storage Management** - `df`, `du`, `fdisk`, `lsblk`, `mount` — disk kitni bhari hai, naya partition kaise banate hain, USB mount karna.
* Day 15: 🗜️ **Archiving & Compression** - Files ko zip karna! `tar`, `gzip`, `zip`, `unzip` — bade folders ko chhota karke bhejte hain.

---

### Day 16-20: Automation & Security 🔒

* Day 16: ⏰ **Cron Jobs & Automation** - Kaam automatically schedule karo! `crontab` se daily, weekly, monthly koi bhi kaam apne aap ho sakta hai.
* Day 17: 🏃 **Systemd & Services** - Linux mein services (programs) kaise chalti hain? `systemctl start/stop/enable` — services control karna seekho.
* Day 18: 🛡️ **Firewall & Security** - `ufw`, `iptables` se computer ko protect karo. Kaunse ports band karne chahiye, kaunse khule — security basics.
* Day 19: 📋 **Log Management** - Computer ki "diary" — `/var/log/` mein sab kuch likha hota hai. `journalctl`, `tail -f`, `grep` se problems dhundhna.
* Day 20: 🌿 **Environment Variables** - `PATH`, `HOME`, `.bashrc`, `.bash_profile` — yeh kya hote hain aur kaise apne hisaab se set karte hain.

---

### Day 21-25: Advanced Linux 🧠

* Day 21: 🗂️ **Linux Filesystem Hierarchy Standard (FHS)** - `/bin`, `/sbin`, `/usr`, `/opt`, `/proc`, `/dev` — har folder ka kya kaam hai, puri samajh.
* Day 22: 📦 **Package Building** - Apna khud ka `.deb` ya `.rpm` package kaise banate hain — developers ke liye zaroori!
* Day 23: 🧬 **Kernel & Boot Process** - Computer on hone ke baad kya hota hai? BIOS → GRUB → Kernel → Systemd — poori story step by step.
* Day 24: 📊 **Performance Monitoring** - System slow kyun hai? `vmstat`, `iostat`, `sar`, `free`, `uptime` — performance diagnose karna.
* Day 25: 🔗 **Linux Networking Advanced** - DNS kaise kaam karta hai, `/etc/hosts`, routing tables, `tcpdump`, `nmap` basics — network troubleshooting.

---

### Day 26-30: Real World Linux 🌍

* Day 26: 🐳 **Docker on Linux** - Docker kya hai aur Linux mein kaise install aur use karte hain? Containers ka concept aur basic Docker commands.
* Day 27: 🔐 **Linux Hardening & Security Best Practices** - SSH hardening, fail2ban, sudo rules, file permissions audit — production server ko secure karna.
* Day 28: 💼 **Backup & Recovery** - Data ka backup kaise lete hain? `rsync`, `cron` + backup scripts, aur system restore karna.
* Day 29: ☁️ **Linux in Cloud (AWS/Azure)** - Cloud mein Linux VM kaise banate hain, SSH se connect karna, web server chalana — real DevOps ka pehla kadam.
* Day 30: 🏆 **Real World Capstone Project** - 30 din mein jo seekha usse ek complete project mein lagate hain — ek automated Linux server setup script jo sab kuch khud kare!
